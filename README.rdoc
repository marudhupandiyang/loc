== README

Gets Location of the User


* Get location via Rails

* Get location via JavaScript 

* Get location via JavaScript HTML5 GeoLocation

How to use

* Clone Repo using
  $> git clone git@github.com:marudhupandiyang/loc.git
   
* Bundle install
  $> cd loc
  $> bundle install
  
* Run rails Server
  $> rails s
  
* Point your favourite browser to http://127.0.0.1:3030
